## Installation

Clone the repository:

```bash
git clone https://github.com/shristhi-2908/react-native-signin-ui.git
```

Navigate to the project folder:

```bash
cd react-native-signin-ui
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npx expo start
```
